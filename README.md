# CampusFind – Lost & Found Campus Board

CampusFind is an Android application designed to help students report and find lost items on campus.

## Features
- Add Lost Items
- Add Found Items
- View All Campus Posts
- My Posts (Edit/Delete your posts)
- Request System to claim items
- Accept / Reject Requests

## Technologies Used
- Java
- Android Studio
- SQLite Database
- RecyclerView

## How It Works
Users can post lost or found items. Other users can view posts and send requests by describing the item. The owner can accept or reject requests.
